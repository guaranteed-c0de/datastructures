Project Title: FinalProjectBST
##This project supports a BST Map, that supports string-int combinations in key-value pairs.
Custom-made Node structure and member functions. BST supports searching, inserting, and deleting nodes.
Source code is Tree.h (header file), Tree.cpp (function definitions), and BSTMap_Driver.cpp (implementation/testing file)

Dependencies: C++11 or higher, CMake 3.16 or higher, Standard C++ library
To download: ```bash
git clone https://github.com/GUARANTEED-C0DE/DataStructures.git
cd FinalProjectBST
mkdir build
cd build
cmake ..
cmake --build .
